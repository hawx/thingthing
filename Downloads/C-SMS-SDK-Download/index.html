C++ SMS SDK Download

The C++ SMS SDK provides you with example code to get you started using the ActiveX SMS API to access your Esendex account. There are two versions of the C++ SMS SDK:

  • C++ 7 SMS SDK
  • C++ 6 SMS SDK

Disclaimer

The sample code provided is not production ready and is only intended as reference information only. Whilst we have tried to make the examples detailed, we have not included examples of everything Esendex offers. Information on the full list of service features can be found in the ActiveX SMS API.
