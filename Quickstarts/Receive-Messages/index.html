Receiving SMS with our API

We provide two ways to receive SMS:

Pull - Receive SMS by polling your Inbox

You can poll our server and retrieve new Messages at a time that suits you.

  • Receive SMS via Pull quickstart

Push - Receive SMS notifications directly to your server

We can push Messages to your server as they arrive with us.
