APIs

The Esendex SMS APIs give developers the ability to send and receive messages directly from their applications.

Each API exposes a series of Internet delivered services conforming to REST, SOAP or HTTP Form Post specifications. Developers who are comfortable with these technologies are able to code directly against these APIs.

The three main APIs are:

  • Esendex REST API
  • Esendex SOAP API
  • HTTP Form Post

For a comparison of the different features available with each API, see the API Comparison page.

Esendex REST API

Our new Esendex REST API is now live. If you want access to the latest functionality, including Voice SMS then this is the place to look.

Sample code

We also provide SMS SDKs for the major development languages/environments. These contain sample code, reusable libraries and tips for using the Esendex service within your applications.
